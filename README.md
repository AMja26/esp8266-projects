# esp8266-projects

This is my first repository to track my ESP8266 learning projects

## Projects

### 1. Blinking LED
My first project was learning how to turn an LED on and off continuously

**Components:** NodeMCU ESP8266, 1 LED, 1 resistor, 2 jumper wires, 1 breadboard

**What I learned:** Basic circuit wiring, digitalWrite, setup(), and loop() functions

### 2. Breathing/Fading LED
This project was an upgrade of the previous. I learned to make the LED fade between its on and off phases

**Components:** NodeMCU ESP8266, 1 LED, 1 resistor, 2 jumper wires, 1 breadboard

**What I learned:** PWM, analogWrite, for loops, PWM and duty cycles, analog vs digital signals
